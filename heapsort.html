Heap Sort

Description

     Heap Sort is an iterative in-place sorting method built on top of heap-based auxiliary data structures. It is less effective than an algorithm with a same time complexity, and data structures with few pieces are not a good fit for it.

The heap is a binary tree-like data structure where each node has a value greater than or equal to that of its offspring. Therefore, the root will contain the highest value.

The data structure is initially arranged to create the heap, and then it is gradually rearranged using an algorithm similar to Selection Sort, beginning with the larger pieces.

Time & Space Complexity

Best Case O(n logn)
Average Case O(n logn)
Worst Case O(n logn)
Space Complexity O(1)
